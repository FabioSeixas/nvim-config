## nvim-config (WIP)
Let's put it here and save time

## What is this about
I'm using [LunarVim](https://github.com/LunarVim/LunarVim) for some time and would like to:

1. have a fast and better way to make my development environment ready on a new machine or setup
2. learn on how to config my own environment
3. have a more tailored, minimal and portable nvim config


## This is not an authentic nvim config
It's done following [this YT series](https://www.youtube.com/watch?v=Vghglz2oR0c) by [chris@machine](https://github.com/ChristianChiarulli).

It's a working in progress.

It has small changes from the original one based on my personal preferences.
